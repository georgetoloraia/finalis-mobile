# Branding

`finalis-core` branding is monochrome-first, geometric, and minimal. Keep the assets in `currentColor` unless the file already specifies a fixed black or white field.

The symbol represents multiple participants converging into one finalized state. That meaning should remain legible in every use.

## Assets

- `finalis-symbol.svg`: core symbol mark
- `finalis-logo-horizontal.svg`: primary logo for README, docs, and site headers
- `finalis-logo-stacked.svg`: stacked logo for square or narrow layouts
- `finalis-favicon.svg`: favicon and small browser/tab usage
- `finalis-splash-lockup.svg`: mobile or launch splash lockup
- `finalis-token-badge.svg`: token-style badge for asset listings
- `finalis-app-icon.svg`: app and wallet packaging icon

## Usage

- Use the symbol only for favicon, wallet surfaces, and repo avatars.
- Use the horizontal logo for README, documentation, and header placements.
- Use the stacked logo for square layouts.
- Use the token badge where an asset-style emblem is needed.
- Use the app icon for packaged applications.

## Rules

- Preserve `currentColor` behavior.
- Do not add gradients.
- Do not add glow effects.
- Do not add 3D styling.
- Do not add decorative outlines.
- Do not redraw proportions, stroke weights, or spacing without replacing the package intentionally.
